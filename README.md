# Dishhub

Dishhub is a simple recipe management app built with React and a JSON Server backend.

## Features

- Browse recipes
- Add new recipes
- Responsive design

## Run Locally

1. Clone the repository:

git clone https://github.com/JustinM094/DishHub

Install dependencies:

- npm install

Start the app:

- npm start

Technologies

- React
- JSON Server
- CSS
